Riddle Game- 
This app is a game of solving riddle. A person will appear when you get 6 wrong letter guesses
GIVE UP - goes to scoreboard
NEW GAME - resets the game
START -starts the game
CLICK LETTER BUTTONS TO CHOOSE LETTERS 
red - wrong letter
green - correct letter 

NOTE: Run best on Android Tablet
